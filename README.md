# Zalgo-rs

A Zalgoifier for rust.

## Getting Started

WIP

### Installing

Add the following to your `Cargo.toml` file:

```toml
[dependencies]
zalgo = { git = "https://github.com/adumbidiot/zalgo-rs" }
```

and to the top of your `main.rs`:

```rust,ignore
extern crate zalgo;
```
